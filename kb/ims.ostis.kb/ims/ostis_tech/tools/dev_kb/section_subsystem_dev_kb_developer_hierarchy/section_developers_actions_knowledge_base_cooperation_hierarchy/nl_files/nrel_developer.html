разработчик* - это бинарное отношение, связывающее какой-либо раздел базы знаний ostis-системы (в пределе – всю базу знаний) и sc-элемент, обозначающий персону, которая может быть разработчиком данного раздела базы знаний, т.е. выполнять проектные задачи в рамках данного раздела. Помимо эксплуатации ostis-системы разработчик* может вносить предложения по изменению какой-либо части базы знаний, оставлять комментарии к предложениям по редактированию базы знаний.
