Алгоритм работы Абстрактного sc-агента верификации scp-программ сводится к поиску некорректностей в рамках scp-программы на основе определений, соответствующих различным классам таких некорректностей, а также посредством запуска соответствующих данным классам некорректностей scp-программ поиска некорректности в scp-программе*. Результатом работы Абстрактного sc-агента верификации scp-программ является формирование в sc-памяти структур, описывающих некорректности в исследуемой scp-программе, если таковые имеются.
