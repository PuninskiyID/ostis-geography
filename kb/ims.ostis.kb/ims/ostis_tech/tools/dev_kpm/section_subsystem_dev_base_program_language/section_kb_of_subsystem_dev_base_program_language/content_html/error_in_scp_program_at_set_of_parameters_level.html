Каждая ошибка в scp-программе на уровне множества параметров описывает некорректный фрагмент, для выявления которого достаточно анализа параметров некоторой scp-программы , т.е. явным образом выделенных аргументов действия (scp-процессе), соответствующего данной scp-программе. К такого рода ошибкам относятся, например, неверное указание ролей этих аргументов в рамках данного действия.
