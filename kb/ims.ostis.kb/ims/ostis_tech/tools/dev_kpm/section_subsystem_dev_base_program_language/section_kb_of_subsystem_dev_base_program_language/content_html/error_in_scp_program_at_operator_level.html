Каждая ошибка в scp-программе на уровне оператора описывает некорректный фрагмент, для выявления которого достаточно анализа оного конкретного scp-оператора , при этом не важно, в состав какой scp-программы он входит. К такого рода ошибкам относится, например, факт указания количества операндов scp-оператора , не соответствующего спецификации соответствующего класса scp-операторов .
