SCn-code core represents a certain way of formatting any sc.s-texts .

Formatting rules:

  1. For the given sc.s-text select key described sc-elements and their names.
  2. Many sentences given sc.s-text divide into groups in accordance with the selected key sc-elements (obviously, in each such sentences must include the name of the corresponding key sc).
  3. Each selected group ofsc.s-sentences purchasedsc.n-article, the title of which is bold the name of the corresponding sc-element, decorated as a special sc.s-sentenceconsisting of only one name and a limited separator dual point comma (like any sc.s-sentence).
  4. Every sc.s-sentencesincluded with some sc.n-articles record on a new line (starting with the first character of the line).
  5. Increase the interval between different sc.n-articles.
  6. All occurrences of the ID specified in the header of the corresponding sc.n-articleofsc.s-sentenceselect a font.
