SCg-код - один из возможных способов визуального представления sc-текстов (далее sc-текст, отображенный с помощью SCg-кода, будем называть sc.g-текстом). Основным принципом, заложенным в основу SCg-кода является то, что каждому sc-элементу в соответствие ставится sc.g-элемент (графическое отображение). Другими словами, любому изображаемому на экране sc.g-элементу соответствует некоторый sc-элемент в базе знаний. В рамках SCg-кода выделено несколько его расширений.
