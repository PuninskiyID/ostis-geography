SCg-код является не только способом визуального представления sc-текстов, а также используется в графическом пользовательском интерфейсе для организации диалога пользователя с системой. Конечно пользователь может производить общение с помощью трех первых уровней (пользователь рисует сообщения вручную). Чтобы повысить эффективность диалога, в третьем расширении SCg-кода вводятся sc.g-элементы управления. Нажатие на такой объект инициирует некоторое действие, которое он обозначает.

Переход от элементов SСg-кода с первым расширением к SСg-коду с третьим расширением осуществляется по правилам, описанным в таблице.
