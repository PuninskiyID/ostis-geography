Очевидно, что если потребуется добавить еще информации об объекте, то единственным выходом будет его дублирование. Другими словами придется создать еще один узел с идентификатором Луна и связывать новые свойства sc.g-парами с ним. Решить данную проблему можно увеличив контактную площадь узла.
