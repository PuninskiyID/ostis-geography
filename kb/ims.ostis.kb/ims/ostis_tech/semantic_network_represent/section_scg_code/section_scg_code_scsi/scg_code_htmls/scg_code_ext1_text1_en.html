The first extension of an sc.g-code is directed on increase of presentation displayed sc.g-texts by reduction the quantity of represented sc.g-elements. To do this, select the most frequently used types of sc.g-elements and each of them in accordance put some graphics and clearly visible sign.

Images of sc-nodes have two characteristics: const and structural type. For their image uses the following rules:

  • constant sc-nodes are represented as circles, the variables - as squares;
  • structural type of sc-node is represented in a circle (square), which corresponds to the sign of constancy;
