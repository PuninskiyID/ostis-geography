The SCg-code allows to use external identifiers of objects that allows the user to better navigate the displayed information. Some of sc.g-elements can be assigned text identifiers to allow the user to read texts written using SCg-code (sc.g-texts). These identifiers are presented in the knowledge base and they can be two types: the global identifier * or the main identifier * for some language. More details on how identified objects can be read on the identification rules.
