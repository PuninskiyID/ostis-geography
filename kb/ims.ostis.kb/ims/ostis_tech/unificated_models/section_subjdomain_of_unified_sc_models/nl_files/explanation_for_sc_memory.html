sc-память представляет собой с одной стороны общую среду для хранения знаний из базы знаний, а с другой стороны – среду для взаимодействия sc-агентов. При этом каждый sc-агент опирается при работе на некоторые известные ему sc-элементы, хранящиеся в этой памяти. Такие элементы будем назвать ключевыми элементами данного sc-агента.
