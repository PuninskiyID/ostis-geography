Связки отношения ключевые sc-элементы sc-агента* связывают между собой sc-узел, обозначающий абстрактный sc-агент и sc-узел, обозначающий множество sc-элементов, которые являются ключевыми для данного абстрактного sc-агента, то есть данные sc-элементы явно упоминаются в рамках программ, реализующих данный абстрактный sc-агент.
