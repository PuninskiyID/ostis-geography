Для установки текущей реализации платформы на компьютер пользователя под управлением ОС Linux необходимо выполнить следующие действия:

  1. Открыть терминал (Ctrl + Alt + T).
  2. Клонировать репозиторий https://github.com/ShunkevichDV/ostis.git в выбранную пользователем директорию:
    1. Если в системе не установлен git, выполнить в терминале команду sudo apt-get update & apt-get install git.
    2. Перейти в выбранную пользователем директорию - выполнить в терминале команду cd %путь-к-выбранной-директории%.
    3. Выполнить в терминале команду
      git clone https://github.com/ShunkevichDV/ostis.git.
  3. Перейти в директорию scripts склонированного репозитория - выполнить в терминале команду cd ostis/scripts.
    Примечание: чтобы получить более новую и стабильную версию компонента sc-machine, необходимо перед выполнением дальнейших инструкций по установке в файле prepare.sh заменить строку
    clone_project https://github.com/ShunkevichDV/sc-machine.git sc-machine scp_stable
    на строку
    clone_project https://github.com/ShunkevichDV/sc-machine.git sc-machine master
    Однако на данном этапе развития системы в данной версии компонента sc-machine недоступен SCP-интерпретатор.
  4. Установить необходимые зависимости и склонировать необходимые дополнительные репозитории - выполнить в терминале команду ./prepare.sh.
  5. Запустить SCTP-сервер - выполнить в терминале команду ./run_sctp.sh.
  6. Открыть ещё одну вкладку терминала, не закрывая старой (Ctrl + Shift + T).
  7. Запустить HTTP-сервер - выполнить команду ./run_scweb.sh во второй вкладке терминала.
  8. Если все вышеуказанные действия были выполнены успешно, в web-браузере по адресу localhost:8000 будет доступна стартовая страница метасистемы IMS.OSTIS. Иначе, необходимо вернуться к пункту 4.
  9. По завершении сеанса работы с системой закрыть терминал.
Для того, чтобы запустить ранее установленную локальную версию системы на компьютере под управлением ОС Linux, необходимо выполнить следующие действия:
  1. Открыть терминал (Ctrl + Alt + T).
  2. Перейти в директорию scripts репозитория ostis - выполнить в терминале команду cd %путь-к-директории-ostis%/scripts.
  3. Выполнить сборку исходных текстов и проверку на синтаксические ошибки - выполнить в терминале команду ./build_kb.sh. Если блок Errors содержит какие-либо сообщения об ошибках, необходимо произвести соответствующие исправления в исходных текстах и выполнить пункт 3 ещё раз.
  4. Запустить SCTP-сервер - выполнить в терминале команду ./run_sctp.sh.
  5. Открыть ещё одну вкладку терминала, не закрывая старой(Ctrl + Shift + T).
  6. Запустить HTTP-сервер - выполнить команду ./run_scweb.sh во второй вкладке терминала.
  7. Если все вышеуказанные действия были выполнены успешно, в web-браузере по адресу localhost:8000 будет доступна стартовая страница метасистемы IMS.OSTIS.
  8. По завершении сеанса работы с системой закрыть терминал.
