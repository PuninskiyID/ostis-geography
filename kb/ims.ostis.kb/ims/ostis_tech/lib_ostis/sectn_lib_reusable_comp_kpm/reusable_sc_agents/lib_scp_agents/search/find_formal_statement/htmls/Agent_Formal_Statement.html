Задачей агента поиска формальной записи утверждения для заданного знака утверждения является нахождение формальной записи утверждения для заданного знака утверждения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска формальной записи утверждения для заданного знака утверждения. Аргументом запроса является знак утверждения, для которого необходимо найти его формальную запись.
