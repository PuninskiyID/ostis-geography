Задачей агента подсчета 90-ой процентили степени вершины онтологии является подсчет 90-ой процентили степени вершины онтологии (Степень вершины - сумма всех её входящих и исходящих связей) и предназначен для оценки качества онтологии (метрики Ингве-Миллера). Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета 90-ой процентили степени вершины онтологии. Единственным аргументом запроса является онтология. В ходе выполнения агента явно указывается связь с scp-программой поиска всех степеней вершин онтологии, scp-программой подсчета 90-ой процентили и scp-программой генерации ответа для агентов оценки качества.
