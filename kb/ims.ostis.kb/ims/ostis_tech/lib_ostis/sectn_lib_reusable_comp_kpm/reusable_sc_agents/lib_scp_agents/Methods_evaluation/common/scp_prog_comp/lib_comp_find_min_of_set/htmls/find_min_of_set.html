Задачей scp-программы поиска минимального значения из множества является поиск минимального значения из множества элементов. Первым входным параметром является множество чисел, для которого будет осуществлен поиск минимального значения. Вторым входным параметром является sc-ссылка, в которую будет записано минимального значение.
