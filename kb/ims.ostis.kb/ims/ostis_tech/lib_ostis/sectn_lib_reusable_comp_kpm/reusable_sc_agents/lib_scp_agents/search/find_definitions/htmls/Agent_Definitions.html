Задачей агента поиска определения или пояснения для заданного объекта является нахождение определения или пояснения для заданного объекта. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска поиска определения или пояснения для заданного объекта. Аргументом запроса является понятие, для которого необходимо найти определение либо пояснение. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск.
