Задачей агента назначения экспертом является назначение пользователя экспертом* базы знаний. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу назначения экспертом/sc_element> базы знаний. Единственным аргументом запроса является пользователь, которого необходимо назначить экспертом* базы знаний.
