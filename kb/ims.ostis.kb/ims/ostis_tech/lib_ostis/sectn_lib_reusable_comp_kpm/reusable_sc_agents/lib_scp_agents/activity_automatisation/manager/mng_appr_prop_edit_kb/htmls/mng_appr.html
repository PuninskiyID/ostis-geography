Задачей агента утверждения менеджером предложения по редактированию базы знаний является утверждение менеджером предложения исполнителя проектного задания. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей действию. утвердить предложение по редактированию базы знаний от имени менеджера. Единственным аргументом запроса является знак действия. рассмотреть предлагаемого исполнителя проектной задачи.

Возможные результаты работы агента:

  • Если первый аргумент знака действия. утвердить предложение по редактированию базы знаний от имени менеджера не является знаком действия. рассмотреть предлагаемого исполнителя проектной задачи, то генерируется сообщение об ошибке (Wrong 1st parameter! It must be sign of action_to_consider_offered_artist_of_project_task).
  • Если пользователь, инициировавший выполнение агента, не является менеджером, то генерируется сообщение "Access denied. User must be a manager."
  • В случае успешного выполнения агента генерируется связка отношения утверждено, связывающая знак действия. рассмотреть предлагаемого исполнителя проектной задачи и знак менеджера, утвердившего предложение исполнителя проектного задания. Знак действия. рассмотреть предлагаемого исполнителя проектной задачи становится прошлой сущностью, удаляется из текущих процессов развития компьютерной системы и добавляется в историю развития компьютерной системы. Содержимое предложения добавляется в план развития компьютерной системы.
