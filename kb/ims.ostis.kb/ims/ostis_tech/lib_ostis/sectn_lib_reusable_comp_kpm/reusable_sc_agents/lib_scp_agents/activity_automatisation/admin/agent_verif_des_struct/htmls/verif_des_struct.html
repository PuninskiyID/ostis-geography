Задачей агента формирования задания на верификацию предложения является формирование задания на верификацию предложения*, содержащего структуру, описывающую прелагаемое изменение в базе знаний. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей действию формирования задания на верификацию предложения. Первым аргументом запроса является знак действия на рассмотрение предложения по редактированию базы знаний. При необходимости, может быть указан второй аргумент, который является экспертом* к данной предметной области. Этот эксперт* будет назначен исполнителем* и должен будет произвести верификацию.
