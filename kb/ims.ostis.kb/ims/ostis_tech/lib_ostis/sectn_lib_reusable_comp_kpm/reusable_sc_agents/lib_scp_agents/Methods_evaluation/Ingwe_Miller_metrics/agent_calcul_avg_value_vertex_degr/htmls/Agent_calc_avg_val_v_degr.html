Задачей агента подсчета средней степени вершины онтологии является подсчет средней степени вершины онтологии (Степень вершины - сумма всех её входящих и исходящих связей) и предназначен для оценки качества онтологии (метрики Ингве-Миллера). Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета средней степени вершины онтологии. Единственным аргументом запроса является онтология. В ходе выполнения агента явно указывается связь с scp-программой поиска всех степеней вершин онтологии, scp-программой подсчета среднего значения и scp-программой генерации ответа для агентов оценки качества.
