Задачей агента удаления sc-элемента является корректировка базы знаний путём удаления выбранных sc-элементов. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу на удаление sc-элемента. Аргументом запроса является sc-элемент, который необходимо удалить из базы знаний.
