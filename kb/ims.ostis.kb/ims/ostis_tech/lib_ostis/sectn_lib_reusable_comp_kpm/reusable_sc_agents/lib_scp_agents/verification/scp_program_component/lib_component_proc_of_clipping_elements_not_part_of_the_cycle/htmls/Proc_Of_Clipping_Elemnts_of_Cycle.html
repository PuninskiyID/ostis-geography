Задачей scp-программы отсечения элементов, не являющихся частью цикла является отсечение всех элементов из структуры, которые не являются частью цикла. На вход scp-программе подаются четыре аргумента. Первый аргумент - узел, который является стартовым для обхода по связкам заданного отношения порядка. Второй аргумент - узел, который замыкает цикл. Третий аргумент - структура, которая содержит цикл и элементы, от которых необходимо избавиться. Четвёртый аргумент - отношение порядка, в рамках которого необходимо найти цикл. Данная scp-программа запускается только в том случае, когда стартовый узел не является конечным. Целью данной scp-программы является удаление из структуры, поданной на вход как третий аргумент, всего пути по бинарным дугам от стартого узла к конечному.
