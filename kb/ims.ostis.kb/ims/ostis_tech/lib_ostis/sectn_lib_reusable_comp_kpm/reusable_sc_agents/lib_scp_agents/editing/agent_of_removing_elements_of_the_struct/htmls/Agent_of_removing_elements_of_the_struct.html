Задачей агента удаления элементов структуры является удаление элементов структуры. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу удаления элементов структуры. Единственным аргументом запроса является структура, из которой требуется удалить элементы.
