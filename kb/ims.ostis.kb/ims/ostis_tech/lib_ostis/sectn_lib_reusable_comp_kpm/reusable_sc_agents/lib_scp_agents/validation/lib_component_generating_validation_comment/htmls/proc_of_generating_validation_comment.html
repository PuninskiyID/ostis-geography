Задачей scp-программы создания комментария об ошибке валидации является создание комментария об ошибке валидации. Первым входным параметром данной scp-программы является знак действия. Вторым входным параметром данной scp-программы является сообщение об ошибке, которое требуется добавить. Третий параметр - множесво ответов.
