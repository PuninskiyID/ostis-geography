Задачей агента подсчета среднего числа различных типов входящих связей вершины онтологии является подсчет среднего числа типов входящих связей для заданной вершины онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета среднего числа различных типов входящих связей вершины онтологии. Единственным аргументом запроса является знак онтологии. В ходе выполнения агента явно указывается связь с scp-программой вычисления различных типов входящих связей всех вершин онтологии, scp-программой подсчета среднего значения и scp-программой генерации ответа для агентов оценки качества.
