Задачей агента поиска утверждений об объекте является нахождение утверждений об объекте. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска утверждений об объекте. Аргументом запроса является понятие, о котором необходимо найти утверждения.
