Задачей агента проверки указания ключевых sc-элементов семантических окрестностей в рамках заданной структуры является нахождение семантических окрестностей, у эелементов заданной структуры, у которых не указан ключевой sc-элемент. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки указания ключевых sc-элементов семантических окрестностей в рамках заданной структуры. Единственным аргументом запроса является структура, элементы которой будут проверены на наличие семантической окрестности.
