Задачей агента проверки указания используемых констант для определений понятий является нахождение ключевых элементов, у которых есть определение или пояснение, но нет используемых констант для этих определений или пояснений. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей действия. проверить наличие используемых констант у понятий, имеющих определения. Аргументом действия является структура, для которой необходимо найти понятия, у которых есть определение или пояснение, но нет используемых констант у этих определений или пояснений.
