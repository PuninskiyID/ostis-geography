Задачей агента поиска ключевых sc-элементов для заданной предметной области является нахождение ключевых sc-элементов для заданной предметной области. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска ключевых sc-элементов для заданной предметной области. Аргументом запроса является предметная область, для которой необходимо найти ключевых sc-элементы. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск.
