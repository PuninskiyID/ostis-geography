Задачей агента проверки наличия основных идентификаторов сущности для всех внешних языков является поиск элементов из предложенного контура, у которых нет основных идентификаторов хотя бы на одном из внешних языков системы. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей действию. проверить основной идентификаторов сущности для всех внешних языков. Первым аргументом запроса является знак структуры, элементы которой проверяются. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, который ограничивает область проверки.
