Задачей scp-программы переноса инцидентных элементов с одной дуги на другую является перенос инцидентных элементов с одной дуги на другую. Первым входным параметром данной scp-программы является дуга, из которой будет произведён перенос инцидентных ей элементов. Вторым входным параметром является дуга, в которую будут перенесены элементы, инцидентные первой дуге.
