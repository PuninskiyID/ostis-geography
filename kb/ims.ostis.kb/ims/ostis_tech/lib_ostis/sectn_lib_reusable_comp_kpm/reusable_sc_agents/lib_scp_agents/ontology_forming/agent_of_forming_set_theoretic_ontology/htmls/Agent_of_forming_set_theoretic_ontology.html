Задачей агента формирования теоретико-множественной онтологии является формирование теоретико-множественной онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу формирования теоретико-множественной онтологии. На вход данному агенту подается предметная область, для которой необходимо сформировать теоретико-множественную онтологию.
