Задачей агента поиска примеров для заданного объекта является нахождение примеров заданного объекта. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска примеров для заданного объекта. Аргументом запроса является понятие, для которого необходимо найти примеры.
