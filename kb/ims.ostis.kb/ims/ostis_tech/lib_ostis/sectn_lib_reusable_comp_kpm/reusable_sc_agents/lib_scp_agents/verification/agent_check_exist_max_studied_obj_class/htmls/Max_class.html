Задачей агента наличия максимального класса объектов исследования у предметных областей является поиск предметных областей из предложенного контура, для которых не указан максимальный класс исследования. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки наличия максимального класса объектов исследования у предметных областей. Единственным аргументом запроса является знак структуры, элементы которой проверяются. При необходимости ограничения области видимости агента в качестве второго аргумента запроса подается контекст действия.
