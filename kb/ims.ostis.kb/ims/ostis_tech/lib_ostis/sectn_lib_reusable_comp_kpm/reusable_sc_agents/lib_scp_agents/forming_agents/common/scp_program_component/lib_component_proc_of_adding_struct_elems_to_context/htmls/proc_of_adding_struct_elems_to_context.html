Задачей sscp-программа добавления элементов структуры в контекст является добавление всех элементов структуры в контекст, если элемент структуры также являются структурой, то ее элементы также будут добавлены в контекст. Данная программа работает рекурсивно для всех вложенных структур. Первым аргументов программы является знак структуры, вторым - знак контекста.
