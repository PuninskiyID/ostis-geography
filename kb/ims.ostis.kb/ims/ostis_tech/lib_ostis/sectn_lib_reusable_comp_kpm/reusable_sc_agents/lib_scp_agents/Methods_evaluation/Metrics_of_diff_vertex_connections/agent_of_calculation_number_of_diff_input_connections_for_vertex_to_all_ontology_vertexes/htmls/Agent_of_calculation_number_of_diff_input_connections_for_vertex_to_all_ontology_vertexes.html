Задачей агента вычисления количества вершин с разными типами входящих связей по отношению ко всем вершинам онтологии является вычисление количества вершин с разными типами входящих связей деленное на количество всех вершин онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления количества вершин с разными типами входящих связей по отношению ко всем вершинам онтологии. Единственным аргументом запроса является знак онтологии. В ходе выполнения агента явно указывается связь с scp-программой подсчета числа различных типов входящих связей вершин онтологии, scp-программой подсчета мощности множества и scp-программой генерации ответа для агентов оценки качества.
