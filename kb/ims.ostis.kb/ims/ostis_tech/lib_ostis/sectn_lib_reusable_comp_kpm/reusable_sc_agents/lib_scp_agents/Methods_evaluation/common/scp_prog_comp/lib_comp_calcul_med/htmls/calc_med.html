Задачей scp-программы подсчета медианы является подсчет медианы для заданного множества элементов. Первым входным параметром является множество чисел, для которых будет высчитываться медиана. Вторым входным параметром является sc-ссылка, в которую будет записано значение медианы. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: scp-программой сортировки множества, отношениями длина* и быть больше*.
