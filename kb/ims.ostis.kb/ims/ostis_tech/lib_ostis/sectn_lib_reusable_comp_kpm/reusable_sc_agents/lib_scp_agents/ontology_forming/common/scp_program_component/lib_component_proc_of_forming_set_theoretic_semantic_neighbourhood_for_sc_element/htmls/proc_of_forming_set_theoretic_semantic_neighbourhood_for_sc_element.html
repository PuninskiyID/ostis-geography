Задачей scp-программы формирования теоретико-множественной семантической окрестности sc-элемента является формирование теоретико-множественной семантической окрестности sc-элемента. Первым входным параметром scp-программы является sc-элемент, исследуемый в рамках предметной области. Вторым входным параметром - знак в теоретико-множественной онтологии, соответствующий рассматриваемой предметной области. В ходе программы явно указывается связь со следующими ключевыми узлами: пример', отношение таксономии*, бинарное отношение.
