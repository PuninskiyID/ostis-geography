Задачей scp-программы подсчета 90-ой процентили является подсчет 90-ой процентили для заданного множества элементов. Первым входным параметром является множество чисел, для которых будет высчитываться 90-ая процентиль. Вторым входным параметром является sc-ссылка, в которую будет записано значение 90-ой процентили. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: scp-программой сортировки множества, отношениями длина* и быть больше*.
