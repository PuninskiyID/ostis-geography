Задачей агента подсчета минимальной глубины онтологии является подсчет минимальной глубины онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета минимальной глубины онтологии и предназначен для оценки качества онтологии (метрики глубины). Единственным аргументом запроса является онтология. В ходе выполнения агента явно указывается связь с scp-программой подсчета всех глубин онтологии, scp-программой поиска минимального значения из множества и scp-программой генерации ответа для агентов оценки качества.
