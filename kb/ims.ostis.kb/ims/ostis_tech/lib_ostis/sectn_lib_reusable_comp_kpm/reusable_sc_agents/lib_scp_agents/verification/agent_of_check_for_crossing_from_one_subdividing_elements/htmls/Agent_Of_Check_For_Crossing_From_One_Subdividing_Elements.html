Задачей агента проверки наличия непустого пересечения у элементов одного разбиения является проверка наличия непустых пересечений у элементов одного разбиения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки наличия непустых пересечений у элементов одного разбиения в рамках указанной структуры. Аргументом запроса является структура, которая содержит элементы, предназначенные для проверки. При необходимости ограничения области видимости агента в качестве второго аргумента запроса подается контекст действия.
