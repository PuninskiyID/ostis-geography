Задачей агента поиска понятий, через которые определяется заданное понятие является нахождение понятий, через которые определяется заданное понятие. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска понятий, через которые определяется заданное понятие. Аргументом запроса является понятие, для которого запрашиваются определяющие его понятия.
