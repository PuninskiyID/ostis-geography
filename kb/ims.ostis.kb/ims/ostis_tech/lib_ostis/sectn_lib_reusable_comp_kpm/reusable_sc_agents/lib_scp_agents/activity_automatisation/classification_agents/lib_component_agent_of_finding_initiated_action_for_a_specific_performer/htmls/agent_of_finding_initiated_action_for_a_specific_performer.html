Задачей агента поиска действий, закреплённых за конкретным исполнителем является поиск действий, закрепленных за конкретным разработчиком* с утверждения* менеджера*. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска действий, закреплённых за конкретным исполнителем. Единственным аргументом запроса является исполнитель*, один из разработчиков* базы знаний, за которым могут быть закреплены какие-то действия.
