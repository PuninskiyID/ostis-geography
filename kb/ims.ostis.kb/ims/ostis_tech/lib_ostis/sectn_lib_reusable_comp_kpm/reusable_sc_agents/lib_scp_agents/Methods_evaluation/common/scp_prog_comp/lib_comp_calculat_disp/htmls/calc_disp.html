Задачей scp-программы подсчета дисперсии является подсчет дисперсии для заданного множества элементов. Первым входным параметром является множество чисел, для которых будет высчитываться дисперсия. Вторым входным параметром является sc-ссылка, в которую будет записано значение дисперсии. В ходе выполнения программы явно указывается связь с scp-программой подсчета мощности множества и scp-программой подсчета среднего значения.
