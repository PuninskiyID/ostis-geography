Задачей scp-программы поиска максимального значения из множества является поиск максимального значения из множества элементов. Первым входным параметром является множество чисел, для которого будет осуществлен поиск максимального значения. Вторым входным параметром является sc-ссылка, в которую будет записано максимальное значение.
