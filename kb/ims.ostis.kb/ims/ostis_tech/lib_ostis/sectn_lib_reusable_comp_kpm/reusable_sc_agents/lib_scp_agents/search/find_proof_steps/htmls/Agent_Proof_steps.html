Задачей агента поиска действий, используемых в рамках решения является нахождение всех действий, используемых в рамках решения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска действий, используемых в рамках решения. Аргументом запроса является узел решения задачи, для которого необходимо найти используемые действия.
