Задачей scp-программы вычисления числа различных типов исходящих связей вершин онтологии является вычисление числа различных типов исходящих связей вершин для заданного множества вершин онотологии. Первым входным параметром является знак онтологии, для вершин которой будет высчитываться число различных типов исходящих связей. Вторым входным параметром является множество вершин данной онтологии. Третьим входным параметром является множество, элементами которого будут значения количества различных типов исходящих связей для вершин данной онтологии. В ходе выполнения программы явно указывается связь с scp-программой подсчета мощности множества.
