Задачей scp-программы вычисления числа различных типов входящих связей вершин онтологии является вычисление числа различных типов входящих связей вершин для заданного множества вершин онотологии. Первым входным параметром является множество вершин данной онтологии. Вторым входным параметром является знак онтологии, для вершин которой будет вычислено число различных типов входящих связей. Третьим входным параметром является множество, элементами которого будут значения количества различных типов входящих связей для вершин данной онтологии.
