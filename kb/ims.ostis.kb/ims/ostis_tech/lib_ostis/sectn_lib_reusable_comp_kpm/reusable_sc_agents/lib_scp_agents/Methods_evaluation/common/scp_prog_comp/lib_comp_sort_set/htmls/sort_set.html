Задачей scp-программы сортировки множества является сортировка множества чисел в виде связного списка. Единственным входным параметром является множество чисел, которое будет отсортировано в порядке возрастания. В результате работы scp-программы, все элементы входного множества будут соединены между собой отношением быть больше*. Кроме того, будет сгенерирован указатель на первый и последний элемент списка, а также размер данного списка. В ходе выполнения программы явно указывается связь с scp-программой подсчета мощности множества, отношениями быть больше* и длина*.
