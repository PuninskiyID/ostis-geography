Задачей агента проверки указания шкалы или единицы измерения для значений параметров в рамках заданной структуры является поиск параметров, которые имеют некорректную спецификацию. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки указания шкалы или единицы измерения для значений параметров в рамках заданной структуры. Единственным аргументом запроса является знак структуры, в рамках которой производится поиск параметров, для которых не указаны либо единица измерения, либо шкала.
