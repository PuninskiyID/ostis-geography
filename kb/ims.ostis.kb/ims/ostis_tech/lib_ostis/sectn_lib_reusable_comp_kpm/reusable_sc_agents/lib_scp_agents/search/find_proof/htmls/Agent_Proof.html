Задачей агента поиска sc-текста доказательства для заданного утверждения является нахождение sc-текста доказательства для заданного утверждения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу sc-текста доказательства для заданного утверждения. Аргументом запроса является утверждение, для которого необходимо найти sc-текст доказательства.
