Задачей агента вычисления среднего числа различных типов исходящих связей вершины онтологии является вычисление среднего числа различных типов исходящих связей вершины онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления среднего числа различных типов исходящих связей вершины онтологии. Единственным аргументом запроса является знак онтологии. В ходе выполнения агента явно указывается связь с scp-программой подсчета числа различных типов исходящих связей вершин онтологии, scp-программой подсчета среднего значения и scp-программой генерации ответа для агентов оценки качества.
