Задачей scp-программы добавления раздела в контекст является добавление знака раздела в контекст, если данный узел там отстутствует, а также спецификации раздела, связей с другими разделами (декомпозиция раздела, подраздел, базовая декомпозиция разделов, и т.д). Программа вызывается рекурсивно для других разделов. Первым аргументов программы является знак раздела, вторым - знак контекста.
