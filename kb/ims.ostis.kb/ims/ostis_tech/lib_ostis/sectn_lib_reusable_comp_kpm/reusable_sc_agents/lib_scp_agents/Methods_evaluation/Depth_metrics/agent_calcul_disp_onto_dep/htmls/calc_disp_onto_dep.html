Задачей агента подсчета дисперсии глубины онтологии является подсчет дисперсии глубины онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета дисперсии глубины онтологии и предназначен для оценки качества онтологии (метрики глубины). Единственным аргументом запроса является онтология. В ходе выполнения агента явно указывается связь с scp-программой подсчета всех глубин онтологии, scp-программой подсчета дисперсии и scp-программой генерации ответа для агентов оценки качества.
