Задачей агента подсчета дисперсии глубины онтологии по отношению к средней глубине является подсчет отношения дисперсии к средней глубине онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета дисперсии глубины онтологии по отношению к средней глубине и предназначен для оценки качества онтологии (метрики глубины). Единственным аргументом запроса является онтология. В ходе выполнения агента явно указывается связь с scp-программой подсчета всех глубин онтологии, scp-программой подсчета дисперсии, scp-программой подсчета среднего значения и scp-программой генерации ответа для агентов оценки качества.
