Задачей scp-программы формирования терминологической онтологии для заданной предметной области является создание терминологической онтологии для заданной предметной области. Первым входным параметром является предметная область, для которой будет генерироваться терминологическая онтология. Вторым входным параметром является узел ответа, в который будут занесены все сгенерированные конструкции. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: множествами предметных областей и терминологических онтологий, отношениями онтология*, основной идентификатор*, идентификатор*, а также подмножества множества естественных языков, на которых представлены идентификаторы и основные идентификаторы понятий, принадлежащих терминологической онтологии.
