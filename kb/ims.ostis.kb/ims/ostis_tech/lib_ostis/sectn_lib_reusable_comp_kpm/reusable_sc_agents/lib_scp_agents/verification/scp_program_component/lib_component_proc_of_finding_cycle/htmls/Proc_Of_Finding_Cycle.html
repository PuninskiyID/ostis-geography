Задачей scp-программы проверки наличия циклов у отношения порядка является поиск циклов в рамках области действия scp-программы. В данной scp-прогрымме предусмотренна возможность ограничения области определения до конкретного раздела. Для рядовых разработчиков удобней использовать эту scp-программу в рамках какой-то предметной области, то есть использовав два входных агрумента (первым аргументом является любое отношение порядка, вторым - раздел, в рамках которого происходит поиск цикла). Для администраторов уместней использовать эту scp-программу в рамках всей базы знаний. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: атрибутами 1'.
