Задачей агента поиска теорем заданной sc-онтологии является нахождение теорем заданной sc-онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска теорем заданной sc-онтологии. Аргументом запроса является sc-онтология, в рамках которой необходимо найти все теоремы. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск.
