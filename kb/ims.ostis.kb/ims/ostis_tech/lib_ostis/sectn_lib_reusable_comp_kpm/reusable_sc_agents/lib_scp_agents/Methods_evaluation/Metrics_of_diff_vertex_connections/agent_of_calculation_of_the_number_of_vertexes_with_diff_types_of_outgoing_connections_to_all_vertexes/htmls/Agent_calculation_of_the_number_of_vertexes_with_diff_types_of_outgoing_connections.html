Задачей агента вычисления вычисления количества вершин с разными типами исходящих связей по отношению ко всем вершинам онтологии является вычисление количества вершин с разными типами исходящих связей вершиныданной онтологии по отношению ко всем вершинам онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления количества вершин с разными типами исходящих связей по отношению ко всем вершинам онтологии. Единственным аргументом запроса является онтология. В ходе выполнения агента явно указывается связь с scp-программой вычисления мощности множества и scp-программой генерации ответа для агентов оценки качества.
