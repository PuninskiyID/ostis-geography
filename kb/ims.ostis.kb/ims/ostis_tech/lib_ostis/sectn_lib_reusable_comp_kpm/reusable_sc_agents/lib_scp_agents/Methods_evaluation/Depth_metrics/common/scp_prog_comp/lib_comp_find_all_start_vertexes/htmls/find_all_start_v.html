Задачей scp-программы поиска всех стартовых вершин является поиск всех стартовых вершин онтологии, относительно которых будет проводиться подсчет всех глубин. Первым входным параметром является множество всех вершин онтологии. Вторым входным параметром является сама онтология. Третьим параметром является множество, в которое будут добавлены все стартовые вершны (Прим. в метриках подсчета глубины онтоглогии, стартовая вершина - это вершина, в которой нет входящих связей по любым отношениям таксономии и дугам принадлежности). В ходе выполнения программы явно указывается связь c понятиями квазибинарного отношения и отношения таксономии.
