Задачей scp-программы нахождения количества элементов класса нахождение количества всех элементов данного класса в системе. Первым входным параметром является класс, для которого должно быть найдено количество его элементов. Вторым входным параметром является sc-ссылка, в которой должно содержаться число элементов класса, начальное значение ссылки должно быть 0. Третьим входным параметром является пустое множество, в которое будут заносится подклассы. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: абсолютными понятиями отношение таксономии, квазибинарное отношение.
