Задача: подсчет количества знаков sc-узлов, принадлежащих заданной структуре.

Описание: агент находит все элементы, принадлежащие заданной структуре под атрибутом rrel_struct и вызывает для них scp-программу подсчёта мощности множества без учёта кратных вхождений либо, если используется контекст для ограничения области поиска агента, scp-программу подсчёта мощности множества без учёта кратных вхождений в рамках заданного контекста.

Инициализация: появление в памяти вопросной конструкции, соответствующей запросу посчитать количество знаков sc-узлов в структуре

Аргументы: структура, контекст(необязательный).

Результаты:

  • Если на вход подан корректный аргумент, будет выведено число - количество знаков sc-узлов, принадлежащих заданной структуре.
  • Если на вход подан не корректный аргумент(первый или второй), будет сгенерировано сообщение о некорректном аргументе.
