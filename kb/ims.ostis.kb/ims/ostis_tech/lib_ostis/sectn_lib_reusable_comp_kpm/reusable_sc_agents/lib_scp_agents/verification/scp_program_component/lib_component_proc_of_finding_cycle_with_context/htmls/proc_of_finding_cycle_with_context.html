Задачей scp-программы поиска цикла с заданным контекстом является поиск циклов для отношений порядка, заданных в структуре, в рамках некоторого контекста. Первым входным параметром данной scp-программы является множество непосещённых бинарных дуг, необходимое для повторного прохода по тем же дугам; вторым входным параметром является узел, который является стартовым в рамках вызванной scp-программы. Третьим входным параметром является отношение порядка, которое необходимо проверить на наличие такой смысловой ошибки как цикл. Четвёртым входным параметром является узел ответа, куда заносятся все элементы цикла. Пятым входным параметром является контекс, который ограничивает область видимости scp-программы.
