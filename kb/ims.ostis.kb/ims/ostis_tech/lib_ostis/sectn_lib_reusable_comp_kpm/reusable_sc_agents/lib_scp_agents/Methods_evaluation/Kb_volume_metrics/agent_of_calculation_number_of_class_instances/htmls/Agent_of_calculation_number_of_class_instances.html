Задача: подсчет количества экземпляров у абсолютного понятия и его частных понятий.

Описание: Частным понятием считается понятие, связанное с данным отношением таксономии. Для каждого такого понятия вызывается процедура подсчета количества элементов класса (работает рекурсивно).

Инициализация: появление в памяти вопросной конструкции, соответствующей действию. подсчитать количества элементов класса

Аргументы: абсолютное понятие

Результаты:

  • Если на вход подан корректный аргумент, будет выведено число - количество элементов класса и его подклассов.
  • Если на вход подан не корректный аргумент, будет сгенерировано сообщение о некорректном аргументе.
