Задачей scp-программы поиска всех вершин нижнего уровня является поиск всех вершин, которые находятся на уровень ниже вершины, переданной на вход в качестве параметра. Поиск осуществляется по исходящим связям любого отношения таксономии и дуг принадлежности. Первым входным параметром является вершина, для которой будет найдено множество вершин нижнего уровня. Вторым входным параметром является онтология. Третьим параметром является множество, в которое будут добавлены все вершны нижнего уровня. В ходе выполнения программы явно указывается связь c понятиями квазибинарного отношения и отношения таксономии.
