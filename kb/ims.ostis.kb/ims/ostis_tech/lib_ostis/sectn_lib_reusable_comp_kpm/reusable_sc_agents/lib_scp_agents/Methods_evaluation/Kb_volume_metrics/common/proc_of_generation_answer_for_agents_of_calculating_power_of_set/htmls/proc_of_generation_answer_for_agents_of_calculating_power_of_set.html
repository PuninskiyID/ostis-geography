Задачей scp-программа генерации ответа для агентов подсчета мощности множества является формирование конструкции, отражающей мощность данного элемента. Первым входным параметром является sc-элемент, являющийся классом, для которого должна быть сгенерирована конструкция. Вторым входным параметром является sc-ссылка, в которой должно содержаться число элементов класса. Третьим аргументом является знак действия, инициируемого при срабатывании агента, для которого должна быть сгенерирована конструкция. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: отношениями десятичное число', измерение* и идентификация*.
