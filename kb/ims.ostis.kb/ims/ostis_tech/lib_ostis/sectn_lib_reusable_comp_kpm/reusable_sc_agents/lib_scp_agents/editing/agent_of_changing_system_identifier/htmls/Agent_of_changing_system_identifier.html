Задачей агента изменения системного идентификатора заданного sc-элемента на указанный файл является удаление старого системного идентификатора и установка нового, выбранного пользователем. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу замены системного идентификатора. Первым аргументом запроса является узел, в котором требуется заменить системный идентификатор. Вторым аргументом запроса является файл на который требуется заменить текущий узел.
