Задачей агента поиска инициированных действий, относящихся к определённому классу действий является поиск инициированных действий, относящихся к определённому классу действий. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска действий, относящихся к определённому классу действий. Единственным аргументом запроса является класс действий, для которого предполагается существование конкретных знаков инициированных действий.
