Задачей scp-программы подсчета среднего значения является подсчет среднего значения для заданного множества элементов. Первым входным параметром является множество чисел, для которых будет высчитываться среднее значение. Вторым входным параметром является sc-ссылка, в которую будет записано среднее значение. В ходе выполнения программы явно указывается связь с scp-программой подсчета мощности множества.
