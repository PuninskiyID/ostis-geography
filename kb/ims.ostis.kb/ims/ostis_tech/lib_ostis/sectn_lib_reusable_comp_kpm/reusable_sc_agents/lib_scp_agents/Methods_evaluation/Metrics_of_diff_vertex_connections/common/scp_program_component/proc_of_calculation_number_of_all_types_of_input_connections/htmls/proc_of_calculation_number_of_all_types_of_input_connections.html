Задачей scp-программы вычисления различных типов входящих связей всех вершин онтологии является вычисление числа различных типов входящих связей всех вершин онотологии. Первым входным параметром является множество вершин, для которых будет высчитываться количество различных типов входящих связей. Вторым входным параметром является множество sc-ссылок,при сложении которых получим общее количество различных типов входящих связей онотологии. Третьим входным параметром является знак предметной области. В ходе выполнения программы явно указывается связь с scp-программой подсчета мощности множества.
