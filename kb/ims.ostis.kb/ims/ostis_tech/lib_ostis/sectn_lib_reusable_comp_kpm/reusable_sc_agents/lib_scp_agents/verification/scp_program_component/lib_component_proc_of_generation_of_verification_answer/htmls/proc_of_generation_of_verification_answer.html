Задачей scp-программа генерации ответа для агентов верификации является генерация результата работы агентов верификации. Первым входным параметром является знак действия, в рамках которого работал агент верификации. Вторым входным параметром является класс действия, знак которого указан выше. Третьим входным параметром является некорректная структура, в которую были добавлены фрагменты базы знаний. Четвертым входным параметром является класс некорректной структуры, в которую будет добавлена некорректная структура из третьего параметра. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: отношениями результат* и ответ*.
