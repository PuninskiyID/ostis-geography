Задачей агента формирования терминологической онтологии для заданной предметной области является построение терминологической онтологии для заданной предметной области. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу формирования терминологической онтологии. Единственным аргументом запроса является предметная область, для которой будет строиться терминологическая онтология.
