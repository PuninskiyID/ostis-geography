Задачей агента проверки наличия определения у понятия является нахождение понятий, не имеющих ни определения, ни пояснения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей действию. проверить наличие определения или пояснения у понятия. Аргументом запроса является структура, в рамках которой необходимо найти понятия, не имеющие ни определения, ни пояснения. При необходимости ограничения области видимости агента в качестве второго аргумента запроса подается контекст действия.
