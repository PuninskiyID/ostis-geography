Задачей scp-программы поиска иллюстраций является поиск иллюстраций для заданного sc-элемента. Первым входным параметром данной scp-программы является сам sc-элемент, для которого необходимо найти все иллюстрации, вторым входным параметром является узел ответа, куда будут занесены все найденные конструкции. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: трансляцией sc-текста* и атрибутом пример'.
