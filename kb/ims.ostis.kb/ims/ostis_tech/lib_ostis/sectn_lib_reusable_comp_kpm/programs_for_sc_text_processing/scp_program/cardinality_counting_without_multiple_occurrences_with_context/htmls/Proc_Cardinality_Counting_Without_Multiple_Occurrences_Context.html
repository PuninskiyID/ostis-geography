Задачей scp-программы подсчёта мощности множества без учёта кратных вхождений в рамках заданного контекста является определение мощности множества без учёта кратных вхождений элементов с учётом контекста. Первым параметром (in-параметр) является множество, мощность которого необходимо определить. Вторым параметром (out-параметр) является sc-ссылка со значением мощности множества. Третьим параметром (in-параметр) является контекст, в рамках которого будет осуществляться поиск элементов множества.
