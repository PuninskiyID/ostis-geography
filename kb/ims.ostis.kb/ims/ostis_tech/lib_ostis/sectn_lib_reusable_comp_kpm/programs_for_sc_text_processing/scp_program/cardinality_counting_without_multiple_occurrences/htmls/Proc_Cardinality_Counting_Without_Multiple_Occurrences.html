Задачей scp-программы подсчёта мощности множества без учёта кратных вхождений является определение мощности множества без учёта кратных вхождений элементов. Первым параметром (in-параметр) является множество, мощность которого необходимо определить. Вторым параметром (out-параметр) является sc-ссылка со значением мощности множества.
