Задачей scp-программы поиска подклассов по отношениям таксономии является поиск подклассов по отношениям таксономии (связанных отношениями таксономии). Первым входным параметром является класс, для которого необходимо найти все подклассы по отношениям таксономии. Вторым входным параметром является множество, куда будет занесен сам класс и все его подклассы по отношениям таксономии.
