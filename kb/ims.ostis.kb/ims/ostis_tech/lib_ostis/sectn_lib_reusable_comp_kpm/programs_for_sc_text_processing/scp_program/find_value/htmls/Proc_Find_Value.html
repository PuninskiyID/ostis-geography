Задачей scp-программы поиска значения является поиск значения величины. На вход данной scp-программе подаётся параметр значения величины и ссылка ответа, куда будет занесён ответ в случае выполнения scp-программы. Первым входным параметром для данной scp-программы является узел, для которого нужно найти значение. Вторым входным параметром является множество, в которое будет помещена построенная цепочка значения. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: абсолютными понятиями величина и число, отношениями значение*, идентификация*, которые используются для поиска цепочки значения. В случае правильного выполнения программы в узел ответ будет занесён результат поиска значения величины.
