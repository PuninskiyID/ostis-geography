Задачей scp-программы поиска идентификаторов с заданным контекстом является поиск основных идентификаторов для заданного sc-элемента в рамках некоторого контекста. Первым входным параметром данной scp-программы является сам sc-элемент, для которого необходимо найти все основные идентификаторы, вторым входным параметром является узел ответа, куда будут занесены все найденные конструкции. Третьим входным параметром является контекст, в рамках которого будет осуществляться поиск. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: основным идентификатором* и множеством языков, на которых идентификаторы могут быть написаны.
