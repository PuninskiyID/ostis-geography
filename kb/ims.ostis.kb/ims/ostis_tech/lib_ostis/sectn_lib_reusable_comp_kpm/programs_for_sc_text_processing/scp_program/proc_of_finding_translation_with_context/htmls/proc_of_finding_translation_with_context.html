Задачей scp-программы поиска трансляций является поиск трансляций для заданного sc-элемента. Первым входным параметром данной scp-программы является сам sc-элемент, для которого необходимо найти все трансляции, вторым входным параметром является узел ответа, куда будут занесены все найденные конструкции. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: трансляцией sc-текста*, множеством языков, на которых идентификаторы могут быть написаны, и атрибутом пример'.
