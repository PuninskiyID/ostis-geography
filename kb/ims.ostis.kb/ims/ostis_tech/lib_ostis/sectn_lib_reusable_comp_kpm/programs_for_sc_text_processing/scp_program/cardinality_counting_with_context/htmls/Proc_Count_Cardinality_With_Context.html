Задачей scp-программы пподсчёта мощности множества в рамках заданного контекста является подсчет мощности заданного множества в рамках заданного контекста. Первым входным параметром (in-параметр) является множество, мощность которого будет посчитана. Вторым входным параметром (in-параметр) является sc-ссылка, в которую будет записано значение мощности множества. Третьим параметром (in-параметр) является контекст, в рамках которого будет осуществляться поиск элементов множества.
