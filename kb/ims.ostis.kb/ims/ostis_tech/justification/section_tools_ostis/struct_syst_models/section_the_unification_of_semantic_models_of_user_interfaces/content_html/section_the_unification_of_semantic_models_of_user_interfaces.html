Пользовательский интерфейс компьютерной системы, построенной на основе предлагаемой технологии, будем рассматривать как специализированную ostis-систему, построенную по той же технологии и предназначенную для трансляции адресуемых пользователю сообщений с внутреннего абстрактного семантического языка представления знаний (SC-кода) на тот или иной внешний язык, тексты которого отображаются пользователю в удобном для него виде, а также для трансляции пользовательских сообщений с внешнего языка внутренний семантический язык компьютерной системы (т.е. в SC-код).

Трактовка пользовательских интерфейсов как компьютерных систем и унификация семантических моделей таких систем дает возможность:

  1. унифицировать проектирование пользовательских интерфейсов;
  2. легко наращивать возможности пользовательских интерфейсов;
  3. легко интегрировать пользовательские интерфейсы с предметными (основными) компьютерными системами;
  4. неограниченно использовать базу знаний предметных компьютерных систем для семантического анализа и понимания вводимой пользователем информации (в частности, естественно-языковых текстов).
