В рамках базовой модели обработки текстов SC-кода выделяются дополнительные классы структур (sc-конструкций) на работу с которыми ориентированы те или другие классы scp-операторов.

С точки зрения данной модели, структура – это минимальная единица SC-кода, обработка которой описывается каким-либо классом scp-операторов.
