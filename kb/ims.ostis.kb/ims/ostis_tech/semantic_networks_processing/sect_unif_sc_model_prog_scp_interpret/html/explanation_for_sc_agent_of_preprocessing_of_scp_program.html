sc-агент предварительной обработки scp-программ произвольного вида осуществляет генерацию дополнительных конструкций, описывающих корректную scp-программу и необходимых для оптимизации процедуры создания scp-процессов, соответствующих данной scp-программе.

Данный агент активируется при появлении в sc-памяти scp-программы, принадлежащей классу корректных scp-программ.

В результате работы данного sc-агента scp-программа попадает во множество подготовленных scp-программ.
