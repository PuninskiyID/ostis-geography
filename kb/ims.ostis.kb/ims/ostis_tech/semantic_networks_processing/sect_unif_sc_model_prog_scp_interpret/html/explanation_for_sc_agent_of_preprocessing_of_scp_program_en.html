sc-agent of preprocessing of any type scp-program realizes generation of the additional constructions, describing a correct scp-program and essential for optimization of the procedure creation of scp-processes, corresponding to this scp-program.

This agent is activated in case of appearance in sc-memory scp-program, belonging to a class of the correct scp-program.

As a result of operation of this sc-agent scp-program gets to a set prepared scp-programs.
