Каждая связка отношения переменные scp-программы* связывает sc-узел, обозначающий scp-программу, и множество, содержащее все sc-элементы, которые входят в какой-либо из scp-операторов данной программы как scp-переменная'. Указанное множество является канторовским, то есть не содержит повторяющихся элементов.
