Операторы класса scp-оператор удаления трехэлементной структуры описывают действие удаления одного или нескольких sc-элементов, составляющих трехэлементную структуру. Каждый оператор данного класса содержит три операнда. При выполнении операторов данного класса выполняется поиск sc-элементов, полностью аналогичный поиску в scp-операторах поиска трехэлементных структур, после чего выполняется удаление значений операндов, помеченных как удаляемый sc-элемент'. Если условиям поиска удовлетворяет несколько имеющихся в памяти структур, то будут удалены элементы только одной из них.
