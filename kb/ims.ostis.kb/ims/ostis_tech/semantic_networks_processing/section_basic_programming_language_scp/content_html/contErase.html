Операторы класса scp-оператор удаления содержимого файла описывают действие удаления произвольного содержимого некоторого файла.

Каждый оператор данного класса содержит один операнд. Данный операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, содержимое которого будет удалено. После выполнения оператора указанная файл будет считаться файлом, для которого не сформировано содержимое, то есть не существует соответствующего ему файла. При этом сам файл не удаляется.
