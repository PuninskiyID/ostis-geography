Операторы класса scp-оператор вычисления длины строкового содержимого файла описывают действие вычисления длины строкового содержимого файла, являющегося значением второго операнда.

Каждый оператор данного класса содержит два операнда.

Первый операнд может иметь произвольный тип значения scp-операнда'. Значением данного операнда является файл, идентифицирующий число, являющееся длиной строкового содержимого, задаваемого вторым операндом. В случае, если данный операнд помечен как scp-операнд со свободным значением', то файл будет сгенерирован, в противном случае будет изменено содержимое уже имеющугося файла.

Второй операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующая строку, длина которой вычисляется.
