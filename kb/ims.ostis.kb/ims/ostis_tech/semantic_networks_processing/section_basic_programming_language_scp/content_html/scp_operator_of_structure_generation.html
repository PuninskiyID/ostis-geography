Операторы класса scp-оператор генерации структур описывают действия генерации каких-либо структур. В результате выполнения операторов данного класса в памяти системы появляются новые сгенерированные sc-элементы. Операторы класса не предполагают ветвления программы в зависимости от выполнения дополнительных условий, поэтому указание следующих операторов осуществляется только при помощи отношения следующий оператор*, без подмножеств.
