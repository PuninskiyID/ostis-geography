Операторы класса scp-оператор получения части строкового содержимого файла по индексам описывают действие получения части строкового содержимого файла, являющегося значением второго операнда, по двум значениям числового содержимого двух файлов, являющихся значениями третьего и четвертого операнда.

Каждый оператор данного класса содержит четыре операнда.

Первый операнд может иметь произвольный тип значения scp-операнда'. Значением данного операнда является файл, идентифицирующий строку, часть строкового содержимого второго операнда, ограниченную индексами. В случае, если данный операнд помечен как scp-операнд со свободным значением', то файл будет сгенерирован, в противном случае будет изменено содержимое уже имеющегося файла.

Второй операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующий строку, часть которой требуется получить.

Третий операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующий число, являющееся индексом - левой границей запрашиваемой подстроки.

Четвертый операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующий число, являющееся индексом - правой границей запрашиваемой подстроки.
