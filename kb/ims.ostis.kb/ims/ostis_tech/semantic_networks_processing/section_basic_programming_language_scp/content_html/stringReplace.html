Операторы класса scp-оператор замены определенной части строкового содержимого файла на содержимое указанной файла описывают действие поиска строкового содержимого файла, являющегося значением третьего операнда, внутри содержимого файла, являющегося значением второго операнда, и замены найденной подстроки строковым содержимым файла, являющегося значением четвертого операнда.

Каждый оператор данного класса содержит четыре операнда.

Первый операнд может иметь произвольный тип значения scp-операнда'. Значением данного операнда является файл, идентифицирующий строку, являющейся результатом поиска и замены найденной подстроки указанной строкой, значения которых определены стркоовым содержимым третьего и четвертого операнда, в строковом содержимом второго операнда. В случае, если данный операнд помечен как scp-операнд со свободным значением', то файл будет сгенерирован, в противном случае будет изменено содержимое уже имеющугося файла.

Второй операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующий строку, в которой осуществляется поиск и замены.

Третий операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующий строку, вхождения которой в содержимое второго операнда требуется заменить.

Четвертый операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующий строку, на которую требуется заменить вхождения содержимого третьего операнда в содержимое второго.
