Параметры типа in-параметр' хоть и соответствуют переменным scp-программы', не могут менять значение в процессе ее интерпретации. Фиксированное значение переменной устанавливается при создании уникальной копии scp-программы ( scp-процесса) для ее интерпретации, и, таким образом, соответствующая scp-переменная' на момент начала ее интерпретации становится scp-константой' в рамках каждого scp-оператора, в котором встречалась данная scp-переменная'.

Использование in-параметров можно рассматривать по аналогии с использованием варианта механизма передачи по значению в традиционных языках программирования, с тем условием, что значение локальной переменной в рамках дочерней программы не может быть изменено.
