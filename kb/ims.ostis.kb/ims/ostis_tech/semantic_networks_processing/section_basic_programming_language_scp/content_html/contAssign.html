Операторы класса scp-оператор копирования содержимого файла описывают действие копирования произвольного содержимого одного файла в содержимое другого файла.

Каждый оператор данного класса содержит два операнда.

Первый операнд может иметь произвольный тип значения scp-операнда'. Значением данного операнда является файл, в которую будет записано содержимое в результате копирования. В случае, если данный операнд помечен как scp-операнд со свободным значением', то файл будет сгенерирован, в противном случае будет изменено содержимое уже имеющугося файла.

Второй операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, из которой будет взято содержимое для копирования.
