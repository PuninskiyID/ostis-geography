Операторы класса scp-оператор вывода информации в консоль применяются при ручной отладке scp-программ в случае отсутствия необходимых для этого средств в используемой среде программирования. Также операторы класса могут использованы для ведения системного протокола, хранимого вне sc-памяти на внешнем языке для выявления некоторых системных ошибок.

Для работы операторов данного класса используются стандартные средства операционной системы, обеспечивающие вывод текстовой информации в консоль.
