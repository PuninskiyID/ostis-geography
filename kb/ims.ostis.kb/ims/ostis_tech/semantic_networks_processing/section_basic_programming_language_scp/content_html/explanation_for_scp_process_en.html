Under an scp-process is understood some structure in sc-memory, single-digit describing the specific act of execution of some scp-program for the given basic data. If the scp-program describes algorithm of the solution of any task in a general view, то scp-process describes specific implementation of this algorithm for the given input parameters.

In fact, scp-process represents the unique copy of a great number of scp-program's operators, in which each sc-variable, except for scp-variables', corresponds the generated sc-constant.
