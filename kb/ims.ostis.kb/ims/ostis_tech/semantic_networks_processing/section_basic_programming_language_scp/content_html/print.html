Операторы класса scp-оператор вывода содержимого файла описывают действие вывода в консоль содержимого заданного файла.

Каждый оператор данного класса содержит один операнд, который должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, содержимое которого должно быть сформировано и будет выведено в консоль. Особенности вывода данных определенного формата определяются возможностями платформы реализации.
