Операторы класса scp-оператор проверки наличия содержимого у файла описывают действие проверки того, имеет ли заданный файл сформированное содержимое. Успешным выполнение считается, если содержимое сформировано. Каждый оператор данного класса содержит один операнд, который должен быть помечен как scp-операнд с заданным значением'. Sc-элемент, являющийся значением операнда должен быть файлом.
