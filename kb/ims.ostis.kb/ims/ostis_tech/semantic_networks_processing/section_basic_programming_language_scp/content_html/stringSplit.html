Операторы класса scp-оператор разбиения строки на подстроки, описывают действие разбиения строкового содержимого sc-элемента на подстроки, разделенные разделителем.

Каждый оператор данного класса содержит три операнда.

Первый операнд может иметь произвольный тип значения scp-операнда'. Значением данного операнда является файл, идентифицирующий множество подстрок строки, задаваемой вторым операндом. В случае, если данный операнд помечен как scp-операнд со свободным значением', то файл будет сгенерирован, в противном случае будет изменено содержимое уже имеющугося файла.

Второй операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующая строку, которая разбивается на подстроки.

Третий операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующая строку, которая выступает как разделитель подстрок второго операнда.
