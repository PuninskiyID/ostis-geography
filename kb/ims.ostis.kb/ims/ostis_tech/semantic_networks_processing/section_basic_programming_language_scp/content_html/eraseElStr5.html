Операторы класса scp-оператор удаления пятиэлементной структуры описывают действие удаления одного или нескольких sc-элементов, составляющих пятиэлементную структуру. Каждый оператор данного класса содержит пять операндов. При выполнении операторов данного класса выполняется поиск sc-элементов, полностью аналогичный поиску в scp-операторах поиска пятиэлементных структур, после чего выполняется удаление значений операндов, помеченных как удаляемый sc-элемент'. Если условиям поиска удовлетворяет несколько имеющихся в памяти структур, то будут удалены элементы только одной из них.
